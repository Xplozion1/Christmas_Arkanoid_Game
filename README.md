Игра в жанре Arkanoid, созданная на движке Unity. Стилистика игры - новогодняя. 

Суть игры: разбить блоки с помощью платформы и летающего мяча. Блоки представлены в виде новогодних атрибутов, так же как и сам мяч. При перезапуске уровня блоки располагаются рандомно. Для победы нужно разбить все блоки, не уронив мяч. 
Управление с помощью WASD, либо стрелочки.

Также я являюсь композитором (проект "Анимур Миядзаки") https://music.yandex.uz/artist/9232452. Оригинальный саундтрек написан мной.
